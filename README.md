# Bump-Bot
Discord.js Server Bump Bot (V12) / Server Advertise Bot

### ✨ Installation
```npm i discord.js parse-ms sequelize sqlite3```

### 🌿 Run
```node index.js```

### 🍂 Dependencies
* discord.js
* parse-ms
* sequelize
* sqlite3
